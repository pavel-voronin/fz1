# Changelog

## 0.1.0 - 2026-03-31

- Initial public release.
- Two-pane terminal UI with tree browsing and fuzzy search.
- Entry selection with optional labeled templates.
- Async description enrichment from `> command` lines.
- Inline editing inside the TUI.
- Empty-catalog onboarding with demo catalog creation.
- Shell integration scripts for zsh, bash, and fish.
